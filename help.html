Help Centre

Learn how to use the VIC Youth Contribution System.

Welcome to the VIC Youth Contribution System.


This system allows administrators, secretaries and members to manage church activities securely.

Approve Member

  1. Open Members.
  2. Select Pending Member.
  3. Click Approve.
  4. The member can now login.

Delete Member

  1. Select Member.
  2. Click Delete.

Add Contribution

  1. Select Member.
  2. Enter Amount.
  3. Click Save Contribution.

Contribution Missing?

  • Verify the correct member.
  • Refresh the page.
